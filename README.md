# Voting Application with CI/CD

A full-stack microservices application demonstrating:
- Frontend (Python Flask)
- Backend Worker (Python)
- Database (PostgreSQL)
- Cache (Redis)
- Results Dashboard (Node.js)
- Complete CI/CD Pipeline with Jenkins

## Architecture
```
User → Voting UI → Redis → Worker → PostgreSQL → Results UI
```

## Technologies
- Kubernetes
- Docker
- Jenkins
- Git/GitHub
- Python
- Node.js
